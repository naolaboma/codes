# LeetCode & Competitive Programming Solutions

A personal collection of solutions to LeetCode problems and competitive programming challenges.

## Structure

- **Problem folders** (e.g., `88-merge-sorted-array/`) — LeetCode solutions organized by problem number, containing a solution file and a `README.md` with the problem statement.
- **Date folders** (e.g., `01-Mar-2025/`) — Daily practice sessions with solutions from that day.
- **A2SV G6 Round folders** — Solutions from A2SV (Africa to Silicon Valley) Group 6 competitive programming rounds.

## Languages

Solutions are primarily written in Python.
